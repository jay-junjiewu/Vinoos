# **App Name**: AquaCraft Showcase

## Core Features:

- Project Gallery: Showcase past fish tank builds with images. Focus on a visually appealing gallery layout.
- Equipment Showcase: Display available fish tank equipment for showcasing key products and services.
- Business Information: Display business hours, contact details, and a Google Maps location link.

## Style Guidelines:

- Primary color: Deep sea blue (#293462) for a professional and calming feel.
- Secondary color: Light teal (#A6D1E6) for accents and highlights.
- Accent: Coral (#FF7F50) to draw attention to key interactive elements.
- Clean and modern typography for readability and a professional look.
- Use simple and elegant icons related to aquariums and fishkeeping.
- Clean and responsive layout that is easy to navigate. Use a grid-based system for the project and equipment galleries.
